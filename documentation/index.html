Gunter - React Documentation

Introduction

Gunter + React — Digital Agency Template. This is created especially for individuals and agencies who provide IT Solutions and Software Services through online. The template contains multiple unique design variations in the home and other pages for the website. Mobile First. Design of the template looks beautiful stunning on all types of screens and devices. Individuals or Agencies who are providing Web, Mobile, eCommerce, AI, Machine Learning or Digital Marketing focused services could love the Gunter template for their existing or next business website!

Setup Development Environment

In order to kickstart the development of the web app with Gunter, the very first thing need to do is to setup a react development environment.

Following tools are needed to setup a react dev environment:

  • Node JS

    • Node JS works as a node package manager behind the scene. It is recommended to download and install the latest version from the Node JS official site http://nodejs.org/.

Preparation

You'll need to install Node.js >=v8.11+ (Recommended Version) (NPM comes along with it) and serve (a NPM package)

Use command line tool i.e. powershell or terminal and navigate to the project(Gunter) root. This is powershell in windows and terminal on mac/linux.

Note:

This project set up with Express.js backend framework, so you have to decide what you do, run with express.js? or without express.js

Let's looking how setup with Express.js & without Express.js:
With Express.js

Just open terminal inside the project root & run: npm install after that move forward one directory such as client cd client and run npm install again that is for client, and again move back to one directory such ass cd .. now run npm run dev and this will return something like the following:

> gunter@1.0.0 dev D:\tf-jsx\gunter

> concurrently "npm run server" "npm run client"

> gunter@1.0.0 server D:\tf-jsx\gunter

> node index.js

> gunter@1.0.0 client D:\tf-jsx\gunter

> npm run start --prefix client

> Read on http://localhost:5000 (It is backend server url with port)

> gunter@0.1.0 start D:\tf-jsx\gunter\client

> react-scripts start

Compiled successfully!

You can now view gunter in the browser.

Local: http://localhost:3000/

On Your Network: http://192.168.1.6:3000/

Now in the browser go to localhost:3000

Without Express.js

Just pick the client folder and inside that open the terminal and run: npm install and this will import all the dependencies for the Gunter React. After that run npm start and this will return something like the following:

Starting the development server...

Compiled successfully!

Local: http://localhost:3000/

On Your Network: http://192.168.1.6:3000/

Now in the browser go to localhost:3000

Fonts

By default, the template loads Montserrat font from Google Web Font Services, you can change the font with the one that suits you best.

You will find the font code in the "style.scss" file: /client/src/assets/css/style.css


@import url('https://fonts.googleapis.com/css?family=Poppins:100,100i,200,200i,300,300i,400,400i,500,500i,600,600i,700,700i,800,800i,900,900i');
